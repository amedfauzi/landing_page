# 🌿 Lumina Terra

This is my first website project from the Foundations Course at [The Odin Project](https://www.theodinproject.com/).

The goal of this assignment was to build a simple static website using only **HTML** and **CSS**.

---

## 📸 Project Overview

This website showcases a collection of nature photography, including:

- 🌊 Beach landscapes
- 🌲 Forest scenery
- 🏜️ Desert views
- ⛰️ Mountain landscapes

The focus of this project was to:

- Practice semantic HTML structure
- Use CSS for styling and layout
- Create a clean and visually appealing design
- Organize content with proper sections and navigation

---

## 🛠️ Built With

- HTML5
- CSS3
- No frameworks
- No JavaScript

---

## 🎯 What I Learned

- Structuring a webpage using semantic elements (`header`, `section`, `footer`, etc.)
- Creating a navigation bar
- Styling layouts using Flexbox
- Managing spacing, typography, and colors
- Organizing project files properly
